# half-wit-cards
Card game based on the original game of Idiot. 
 
 Usage
---
 
Start the development server with this command:
 
```
npm start
```
In a separate terminal tab ```webpack -w```
 
Setup
---
 
```
npm install
```
 
Compile
---
 
```
npm run compile
```
